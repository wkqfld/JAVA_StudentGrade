# JAVA_StudentGrade
- 성적관리 프로그램 만들기 : 과목별 수강 학생들의 점수 / 학점 계산하여 리포트 출력


## Package
  1) grade : 평가방법 관련 클래스 묶음 -> 인터페이스 + 구현클래스
  2) school : 학생, 성적, 과목 클래스 + 이걸 담는 학교 클래스 + 성적표 산출
  3) test: 학교에 성적/학생 리스트 등록 및 리포트 실제 생성 (프론트 역할 겸)
  4) utils : define
